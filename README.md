1. Efectos GSAP recomendados
Global (todas las escenas):

GSAP Observer + ScrollTrigger para:

Fijar el contenedor principal.

Hacer que cada sección “encaje” como slide al hacer scroll.

Animar entrada/salida de izquierda/derecha (texto ↔ visual).

Escena 1:

Texto entra con from({ x: -50, opacity: 0 }).

Tarjeta foil entra con from({ x: 50, opacity: 0, rotate: -3 }).

Efecto foil + elastic drag con onMouseMove + gsap.to y ease: "elastic.out(1, 0.4)".

Escena 2:

Esfera de tecnologías girando con gsap.to infinito (repeat: -1, ease: "none").

Al entrar la escena, pequeño scale de la esfera (fromTo scale 0.8 → 1).

Escena 3:

Cada proyecto:

Fade + slide suave al cambiar (from({ y: 30, opacity: 0 })).

Puedes usar un array de proyectos y animar el cambio con GSAP cuando el Observer detecte nueva sección.

Escena 4:

Certificados con stagger en la entrada (stagger: 0.1).

Hover con gsap.to para scale: 1.03 y box-shadow.

Escena 5:

Formulario entra con from({ y: 40, opacity: 0 }).

Botón de enviar con pequeño hover tipo “pulse” (scale: 1.02, backgroundPosition para simular foil).
